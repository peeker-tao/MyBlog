# 我的博客

一个使用 Node.js、Express、SQLite 和 EJS 构建的简约博客。

## 功能

- 文章管理（新建、编辑、删除）
- Markdown 编辑与实时预览
- 文章列表、详情页与按月归档
- 评论系统与审核
- 分类与标签云
- 关于页面
- 全文搜索与标签搜索

## 启动

1. 安装依赖：
   - `npm install`
2. 运行迁移：
   - `npm run migrate`
3. 初始化示例数据（可选）：
   - `npm run seed`
   - `npm run seed:random`

4. 启动应用：
   - `npm run dev`

## 说明

- 管理后台路由为 `/admin`，用于本地简单管理。
- 全文搜索使用 SQLite FTS5。
